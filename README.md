# YouTube-Chat-Viewer-Tool

Bu proje, YouTube canlı yayınlarının sohbet (chat) kısmını web sitesi üzerinden görüntülemek için geliştirilmiştir.

Siteye girildiğinde, kullanıcıdan yayın URL’si yerine yalnızca **yayın ID (yanındaki URL kısmı)** istenir ve girilen ID’ye ait canlı sohbet ekranda anlık olarak gösterilir.

Proje, geliştirme sürecinde AI araçlarından destek alınarak oluşturulmuştur.

## Özellikler
- YouTube canlı yayın chatini anlık görüntüleme
- Sadece yayın ID girerek kullanım
- Basit ve hızlı arayüz
- Türk bayrağı ve Atatürk temalı görsel dokunuşlar

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Kullanım
1. Web sitesini açın
2. YouTube canlı yayının **ID** kısmını girin
3. Canlı sohbeti anlık olarak izleyin

## AI Kullanımı
Bu proje, geliştirme aşamasında AI destekli araçlar kullanılarak oluşturulmuştur.
